Use the `prompt-eval:prompt-eval` skill to run an evaluation campaign on the profile named `$ARGUMENTS`.

Pass any flags after the profile name (e.g. `--mode auto`, `--max-budget 5`, `--max-rounds 3`) through to the skill.

Refer to the skill at `skills/prompt-eval/SKILL.md` for the full procedure.
